# Security Policy

## Supported Versions

Security updates are provided for the latest App Store release of Speed Demon.

If you are using an older version, please update to the latest release before reporting a security issue.

## Reporting a Vulnerability

Please do **not** report security vulnerabilities in public GitHub issues.

- Support and non-sensitive bug reports:  
  https://github.com/wildwestdev/speed-demon-support/issues/new/choose
- Security vulnerabilities and sensitive disclosures:  
  Email: **security@wildwestbrewing.co**

When reporting a potential security issue, include:

- App version and build number
- Device model and iOS version
- Steps to reproduce
- Expected behavior and actual behavior
- Impact assessment (what data or capability is affected)
- Screenshots or logs (with private data removed)

## Disclosure Process

After receiving a report, we will:

1. Acknowledge receipt.
2. Triage and validate the issue.
3. Work on a fix and release timeline.
4. Publish remediation details in release notes where appropriate.

We appreciate responsible disclosure and coordinated reporting.
