Speed Demon Manual

Manual Version: 1.0 App Focus: iPhone + iPad Sync: SwiftData + CloudKit

Quick Start

  1. Open Speed Demon and grant location permission when prompted.
  2. Create or select a trip.
  3. Press Play to start recording.
  4. Drive normally; distance and session metrics update live.
  5. Press Pause to end the current session.
Only one trip can be active at a time. Starting a new trip automatically pauses any currently active trip.

Trips and Sessions

Trip controls

  • Play/Pause: starts or ends the active session.
  • Reset: resets current session metrics without changing running state.
  • History: opens detailed session history for that trip.
  • Delete: removes the trip and all its sessions.

Session behavior

  • Pressing Play creates a new session with start time and start location.
  • Pressing Pause closes the session with end time and end location.
  • Each trip card shows distance, duration, average speed, and session count.

Settings

About

  • Shows app version/build in x.y.z format.
  • Report an issue opens the public support issue form.

Core options

  • Units: metric or imperial display.
  • Background Tracking: continue updates while backgrounded.
  • Auto-pause delay: 0–60 minutes (0 disables auto-pause).

Advanced thresholds

  • Auto-pause moving speed threshold
  • Sync save interval
  • Cloud refresh interval

Debug Display

  • Shows Motion + Sync status lines when enabled.
  • Can display GPS authorization and accuracy diagnostics.

Sync and Data

Speed Demon stores trip data with SwiftData and syncs via CloudKit private database when available.

Synced Syncing Local fallback
  • Synced: local and cloud state are aligned.
  • Syncing: upload or refresh is in progress.
  • Local fallback: cloud unavailable; local persistence continues.
Use Sync Now in Settings to trigger an immediate sync attempt.

Live Activity and CarPlay

Live Activity

  • Title shows monitored trip name: Speed Demon - TripName.
  • Metrics are shown in two columns:
Left Column
Distance, Speed
Right Column
Duration, Avg Speed

CarPlay notes

  • CarPlay requires Apple-approved entitlement for production use.
  • Without entitlement approval, CarPlay app availability may be limited or hidden.

Import / Export

  • Export trips as JSON or CSV.
  • Import trips from JSON.
  • Trip history can be printed from the history screen.

Recommended backup workflow: export JSON periodically and keep a dated archive.

Troubleshooting

Trip does not auto-pause

  • Check Auto-pause delay is greater than 0.
  • Check moving speed threshold is appropriate for your conditions.

Data not syncing between devices

  • Confirm same Apple ID on both devices.
  • Ensure iCloud Drive and app iCloud permissions are enabled.
  • Use Sync Now from Settings.

Live Activity not updating correctly

  • Pause and restart the trip.
  • Ensure Live Activities are enabled in iOS settings for the app.
